AI English Exam Practice Agent

EAgent is a lightweight AI-powered tutor that helps high school students practice English exams daily.

It demonstrates how an AI agent can use tools + multi-step reasoning to simulate a real learning assistant.

What It Does

EAgent helps students preparing for:

🇻🇳 Vietnamese National High School Exam

🌍 IELTS Basic (≈5.5 – 6.5)

Instead of static exercises, students interact with an AI tutor that guides the full learning cycle.

Key Capabilities

📝 Generate Practice Tests
Creates structured English exam questions.

✅ Automatic Grading
Evaluates student answers instantly.

🔍 Mistake Analysis
Detects grammar and vocabulary weaknesses.

📚 Grammar Explanation
Explains why answers are correct or incorrect.

🎯 Learning Suggestions
Recommends the next exercises based on mistakes.

AI Agent Workflow

The agent performs multi-step reasoning using tools:

Student Request
↓
Generate Exam
↓
Grade Answers
↓
Analyze Mistakes
↓
Explain Grammar
↓
Recommend Next Practice

This simulates a personal AI English tutor.
